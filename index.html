O que é Web Design?

Web Design é a arte de criar e desenvolver sites, criando o código e a arte gráfica do site.

Qual o objetivo de se contratar um Web Designer?

O Web Designer irá ver o seu negócio como um todo e então ira entender qual a melhor forma de site para você.

Por que é importante ter um site para a empresa?

O dite que o web designer desenvolver irá ajudar a sua empresa ter mais visão no mercado.

Processo para a criação
e desenvolvimento

Criação do Design.

Primeiramente foi desenhado um design na ferramenta XD da Adobe, onde se pode acertar todo o design antes de ir para a codificação.

Desenvolvimento Front-End.

O desenvolvimento front-end é onde se prepara o resultado obtido no XD para a codificação, escolhendo as melhores ferramentas e linguagens para o desenvolvimento.

Ferramentas e linguagens.

No desenvolvimento deste portifolio foram usadas as linguagen padrões que são HTML 5, CSS 3 e Javascript. E ferramenta para agilizar o desenvolvimento foi o Bootstrap 4.1.

Conhecimentos necessários.

Lógica de programação, Inglês intermediário para termos técnicos, linguagens e suas estruturas e o conhecimento do framework utilizado.

Ferramenta de Codificação.

Neste projeto é usado o Visual Studio Code da Microsoft que é um editor de código otimizado para criar e examinar códigos em busca de erros.

Finalização.

Após tudo estar funcionando sem erros é hora de escolher um domínio e compra-lo e então podemos partir para colocar o site no ar através de uma empresa de hospedagem de sites.
